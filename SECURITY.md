# Security Policy

## Overview

This is a client-side web application. All data processing happens in the browser -- no data is sent to any server.

## Reporting Vulnerabilities

If you discover a security issue (e.g., XSS, CSP bypass, dependency vulnerability), please email faruk.diblen@resoftlabs.com.

## Scope

This policy covers the web application repository only. The schema and CLI repositories have their own security policies.

## Supported Versions

Only the latest release is actively maintained.
